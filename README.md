# Heart-Disease-Analysis


## Project Overview

This project aims to analyze heart disease data to identify key risk factors and generate meaningful insights using database integration, data visualization, and dashboard development.

The objective is to:
- Extract and manage structured health data from a database
- Perform data preparation and cleaning
- Create analytical visualizations
- Develop an interactive dashboard
- Present business insights through storytelling
- Integrate dashboard into a web interface

---

## Problem Statement

Heart disease is one of the leading causes of mortality worldwide.  
This project analyzes patient health attributes to identify patterns and risk indicators that contribute to heart disease.

---

## Project Workflow

1. Data Collection & Extraction from Database
2. Data Preparation and Cleaning
3. Data Visualization
4. Dashboard Development (Tableau)
5. Story Creation (Insights & Findings)
6. Performance Testing
7. Web Integration
8. Final Documentation & Demonstration
